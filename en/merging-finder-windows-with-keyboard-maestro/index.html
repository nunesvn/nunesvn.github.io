Merging Finder windows with simple Keyboard Maestro Macro

22/Apr 2014 By mk
OSX

The new Finder tabs on OS X 10.9 Mavericks are frequently sub-used. This is a simple Keyboard Maestro macro which:

  • Shows Finder, if it is not already on focus
  • Merge all your Finder windows into one, with tabs

Even if you are not a tabs person, it worth a try. No more losing control of too many Finder windows! :)
